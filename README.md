**Um guia autodidata de 12 meses para dominar IA, Machine Learning e LLMs.**

Este roadmap organiza os conhecimentos necessários para atuar com **Inteligência Artificial moderna**, unindo a base teórica de instituições como MIT e Stanford com as demandas práticas de engenharia de software e MLOps.

---

## Visão Geral do Programa

O cronograma está dividido em **8 fases**, focadas em construir uma base sólida antes de avançar para modelos generativos e infraestrutura.

| Fase | Área de Foco | Meses |
| :--- | :--- | :--- |
| **1** | Python e Data Science | 1–2 |
| **2** | Matemática para Machine Learning | 2–3 |
| **3** | Machine Learning Clássico | 3–5 |
| **4** | Deep Learning | 5–7 |
| **5** | Transformers e LLMs | 7–8 |
| **6** | Engenharia de IA (RAG & Agentes) | 9–10 |
| **7** | MLOps e Deploy | 11 |
| **8** | Projeto Profissional (Capstone) | 12 |

> 🔗 **Versão Interativa:** [Acesse o checklist no Notion](https://www.notion.so/AI-Engineer-Roadmap-2026-3206256ea0a780a5927ee22a652efcb1)

## Tech Stack

Durante o roadmap, você dominará as ferramentas essenciais para um **AI Engineer**:

* **Linguagem:** Python
* **Frameworks de ML/DL:** PyTorch, scikit-learn
* **Ecossistema IA:** HuggingFace, LangChain, LlamaIndex
* **Backend & Infra:** FastAPI, Docker, PostgreSQL
* **Dados:** Vector Databases (ChromaDB, Pinecone, Milvus)

---

## Projetos Incluídos

O aprendizado é validado através da construção de um portfólio real:

* **Análise de Dados:** Insights e visualização de dados complexos.
* **Modelagem Preditiva:** Previsor de preços com Machine Learning.
* **Visão Computacional:** Classificador de imagens com CNN.
* **NLP Moderna:** Chatbot com RAG (Retrieval-Augmented Generation).
* **Automação:** Assistente de IA e Agentes.
* **Personalização:** Sistemas de recomendação.

---

## Como Usar este Roadmap

Para um progresso consistente, recomenda-se:

* **Carga Horária:** 8–12 horas por semana.
* **Estratégia:** `Aprender → Construir → Falhar → Melhorar → Escalar`.
* **Recursos:** Consulte o arquivo `resources.md` para bibliografia, papers e datasets.
