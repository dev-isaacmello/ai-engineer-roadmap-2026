# Recursos de Estudo: Machine Learning & IA

Este guia compila os recursos fundamentais, do nível iniciante ao avançado, para dominar a ciência de dados e a inteligência artificial moderna.

---

## 1. Machine Learning (Fundamentos)
*O foco aqui é entender a mecânica dos algoritmos clássicos antes de avançar para redes neurais.*

* **[Livro] Hands-On Machine Learning** (Aurélien Géron): O guia definitivo. Cobre de Regressão Linear a Transformers usando Scikit-Learn e TensorFlow.
* **[Curso] Machine Learning Specialization** (Andrew Ng / Coursera): A porta de entrada clássica para entender a intuição matemática sem medo.
* **[Livro] Introduction to Statistical Learning (ISLP)**: Essencial para quem deseja uma base estatística sólida aplicada em Python.

---

## 2. Deep Learning & Redes Neurais
*Onde a "mágica" da visão computacional e do processamento de linguagem acontece.*

* **[Curso] Deep Learning Specialization** (DeepLearning.AI): Sequência lógica do curso de ML do Andrew Ng. Cobre CNNs, RNNs e LSTMs.
* **[Livro] Deep Learning** (Ian Goodfellow): Conhecido como "A Bíblia". Focado na teoria rigorosa, álgebra linear e cálculo dos gradientes.
* **[Curso] Fast.ai (Practical Deep Learning for Coders)**: Excelente abordagem "top-down" (codar primeiro, entender a teoria depois).

---

## 3. Matemática & Teoria
*Os pilares fundamentais para não ser apenas um "importador de bibliotecas".*

* **[Livro] Mathematics for Machine Learning** (Marc Deisenroth): Cobre Álgebra Linear, Cálculo e Probabilidade com foco direto em modelos de IA.
* **[Canal] StatQuest with Josh Starmer**: A melhor didática visual para entender conceitos complexos de estatística e ML.

---

## 4. Papers Milestones (Leitura Obrigatória)
*Artigos científicos que mudaram o rumo da indústria.*

1.  **Attention Is All You Need (2017)**: Introduziu os *Transformers*, a base do ChatGPT e Claude.
2.  **Deep Residual Learning for Image Recognition (ResNet)**: Revolucionou a profundidade das redes neurais em visão computacional.
3.  **BERT: Pre-training of Deep Bidirectional Transformers**: Fundamental para entender o processamento de linguagem moderno.

## 6. Datasets & Prática
*Onde encontrar dados e testar suas habilidades.*

* **[Kaggle](https://www.kaggle.com)**: Competições, datasets reais e notebooks da comunidade.
* **[Hugging Face Hub](https://huggingface.co/datasets)**: O "GitHub" dos modelos e datasets de IA modernos.
* **[UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/index.php)**: Repositório clássico para benchmarks acadêmicos.
* **[Papers with Code](https://paperswithcode.com)**: Conecta artigos científicos aos seus respectivos códigos de implementação.

---

> **Dica de Estudo:** Comece pelo curso do **Andrew Ng** para a base teórica e siga com o livro do **Aurélien Géron** para a prática. Não tente ler o livro do Goodfellow de capa a capa logo no início; use-o como consulta técnica.
